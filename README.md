# Museum Web Application

A **prototype museum web application** built with **Python Flask**.  
It demonstrates responsive design, accessibility, usability, and UX principles.  
The app allows users to view exhibits, news, events, services, submit feedback, and contact the museum. Registered users can log in, and future development includes booking visits. Admin functionality (mockup) includes managing news, events, and feedback.

---

## Features

- Home page with overview, featured items, and upcoming events
- Exhibits information page
- News page
- Events page (upcoming & past events)
- Services offered page
- Visitor feedback page with form
- Contact page with form and museum info
- User login and registration (prototype)
- Responsive sidebar navigation
- Accessible and semantic HTML
- Clean, modern UI design
